## üöÄ Key Projects

### ‚ö° **[Turbo-TOSEC](https://github.com/DeponesStudio/turbo-tosec)**
> *High-performance data ingestion & audit engine for big datasets.*

A cutting-edge demonstration of modern **Data Engineering** on local machines. Turbo-TOSEC solves the bottleneck of processing massive DAT files by utilizing a **Map-Reduce** architecture with **DuckDB** and **Parquet**.

* ‚úÖ **Architecture:** Multi-process (Map) -> Parquet Storage -> Bulk Load (Reduce).
* ‚úÖ **Performance:** Processes gigabytes of XML data in seconds, not hours.
* ‚úÖ **Tech:** Python, DuckDB, Pandas, XML Stream Parsing.

[**View Repository**](https://github.com/DeponesStudio/turbo-tosec)

### üìù **[Glyph](https://github.com/berkacunas/Glyph)**
> *A modern, multi-tab Markdown editor built with Python & PySide6.*

Glyph combines the flexibility of web technologies (Chromium) with the raw performance of native code (Qt). It stands as our proof of concept for robust desktop GUI engineering.

* ‚úÖ **Live Preview:** Side-by-side rendering with instant feedback.
* ‚úÖ **Professional:** PDF export, advanced syntax highlighting, distraction-free mode.
* ‚úÖ **Open Source:** Licensed under GPLv3.

[**Download Glyph**](https://github.com/berkacunas/Glyph/releases/latest) ‚Ä¢ [**View Source**](https://github.com/berkacunas/Glyph)
